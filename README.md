# Faculty Review Portal

Faculty Review Portal is a full-stack web application that enables students to review and rate faculty members while providing administrators with tools to manage faculty profiles, users, and reviews. The platform offers secure authentication, detailed faculty ratings, a student discussion forum, and an administrative dashboard for moderation.

## 🚀 Features

### 👨‍🎓 Student

* Register using a **@g.bracu.ac.bd** email address
* Secure login with JWT authentication
* View faculty list and detailed faculty profiles
* Rate faculty based on:

  * Teaching
  * Marking
  * Behaviour
* Automatic overall rating calculation
* Add one review per faculty (duplicate reviews are prevented)
* Edit or delete own reviews
* View average faculty ratings
* Create query posts
* Comment on other students' queries
* View personal profile information

### 🛡️ Admin

* Secure admin login (no public registration)
* Add, edit, and delete faculty profiles
* Manage faculty information, including designation, courses, and educational background
* View all registered users
* Block or delete users
* Hide or remove abusive reviews
* Moderate student-generated content

## 🎯 Purpose

Faculty Review Portal is designed to provide students with a transparent platform for sharing constructive feedback about faculty members while enabling administrators to efficiently manage faculty information, reviews, and user-generated content. The project demonstrates full-stack web development concepts, including authentication, role-based authorization, CRUD operations, relational databases, and secure API development.
